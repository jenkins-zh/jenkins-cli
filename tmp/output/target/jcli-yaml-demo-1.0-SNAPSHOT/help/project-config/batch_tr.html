Projenin yapılandırılması için Windows batch scripti çalıştırır. Script bulunduğu dizini çalışma alanı olarak kabul edecektir. Metin kutusuna girdiğiniz dosya, batch dosyası, çalıştırılacak, ve çalıştırma sonucunda %ERRORLEVEL% değeri 0 olmaz ise yapılandırma başarısız sayılacaktır.

Eğer hali hazırda SCM içerisinde bir batch dosyanız var ise, batch dosyasının yolunu yazmanız yeterli olacak (çalışma alanı dizinine bağlı olan yolu) ve çalıştırılabilecektir.
