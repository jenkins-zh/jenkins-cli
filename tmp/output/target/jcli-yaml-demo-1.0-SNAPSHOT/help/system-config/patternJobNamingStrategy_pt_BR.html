Defina um padrão (expressão regular) para checar se o nome de um job é válido ou não.

Forçando a checagem em jobs que já existem, vai permitir que você reforce uma convenção - e.g. mesmo que os usuários não mudem o nome, O job será validado com o padrão definido em cada submissão e atualização até que o nome confirme.
Essa opção não afetará a execução de jobs com nomes não complascentes. Isto apenas controla o processo de validação no momento de salvar as configurações de um job.
