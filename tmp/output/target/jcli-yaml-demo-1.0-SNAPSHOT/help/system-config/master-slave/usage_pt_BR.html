Controla como o Jenkins agenda as construções nesta máquina.
Utilizar este agente tanto quando possível
Esta é a configuração padrão. Neste modo, o Jenkins usa este agente livremente. Sempre que houver uma construção que possa ser feita usando este agente, o Jenkins usará ele.
Deixar esta máquina apenas para tarefas amarradas
Neste modo, o Jenkins apenas construirá um projeto nesta máquina quando este projeto especificamente tiver este agente como o "nó associado". Isto permite que um agente seja reservado para certos tipos de tarefas. Por exemplo, para executar testes de desempenho do Jenkins contínuamente, você pode usar esta configuração com o número de executores como sendo 1, tal que apenas um teste de desempenho execute a qualquer tempo definido, e que um executor não será bloqueado por outras construções que possam ser feitas em outros agentes.
