Bu kısım, agent'in opsiyonel, okunaklı açıklamasıdır. Bu bilgi projenin konfigürasyon sayfasında gösterilir.

Bazı agent'leriniz, diğerlerinden farklı olabilir. Bu tür durumlarda açıklama alanı, agent'leri diğerlerinden ayırdecek ve farkı gösterecektir. Mesela bir agent'e "Windows agent" tanımı girilmesi, proje sahiplerinin windows üzerinde çalışmak istediklerinde, yapılandırmalarını kolayca bu nod(lar) üzerinde çalıştırabilmesini sağlar.
