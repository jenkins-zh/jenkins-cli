Isto controla o número de construções concorrentes que o Jenkins pode executar neste agente. Assim o valor afeta toda a carga de sistema que o Jenkins pode sustentar. Um bom valor para iniciar seria o número de processadores da máquina.

Aumentando este valor além disso pode causar lentidão na construção, mas também pode aumentar a capacidade total, porque permite a CPU construir um projeto enquanto uma outra construção está aguardando por operação de E/S.

Atribuir 0 a este valor é útil para remover temporáriamente um agente desabilitado do Jenkins sem perder outras informações de configuração.
