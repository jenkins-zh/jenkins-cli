控制 Jenkins 什麼時間啟動及停止 Agent。
盡可能讓 Agent 保持在上線狀態
這是預設值也是標準設定。 此模式下，Jenkins 會盡可以讓 Agent 上線。 如果 Jenkins 可以自已啟動 Agent，在 Agent 無法使用時會每隔一段時間試看看能不能將其重啟。 Jenkins 不會主動讓 Agent 離線。
需要時讓 Jenkins 上線，閒置時離線
此模式下，如果 Jenkins 可以自已啟動 Agent，當有未執行作業符合下列情況時，Jenkins 會每隔一段時間試看看能不能將 Agent 啟動:
  • 作業已經在佇列裡等超過指定的上線延遲時間
  • 作業可以在這個 Agent 上執行
Agent 離線時機:
  • 這個 Agent 沒在建置任何作業
  • Agent 已經閒置超過指定的離線延遲時間
