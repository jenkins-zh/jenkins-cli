Per impostazione predefinita, Jenkins salva tutti i propri dati in questa directory sul filesystem.

Ci sono modi differenti di modificare la directory home di Jenkins:

  • Modificare la variabile JENKINS_HOME nel file di configurazione di Jenkins (ad esempio /etc/sysconfig/jenkins su Red Hat Linux).
  • Utilizzare lo strumento di amministrazione del container Web per impostare la variabile d'ambiente JENKINS_HOME.
  • Impostare la variabile d'ambiente JENKINS_HOME prima di avviare il container Web, o prima di avviare Jenkins direttamente dal file WAR.
  • Impostare la proprietà di sistema Java JENKINS_HOME durante l'avvio del container Web, o durante l'avvio di Jenkins direttamente dal file WAR.
  • Modificare web.xml in jenkins.war (o la sua immagine espansa nel container Web). Quest'opzione non è consigliata.
Questo valore non può essere modificato finché Jenkins è in esecuzione.
