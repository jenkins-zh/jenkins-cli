Jenkins tüm veri dosyalarını, dosya sistemi içerisinde bu dizinde tutar. Bu dizini birkaç yöntemle değiştirebilirsiniz.
  1. Web container'ınızın admin arayüzünü kullanıp, JENKINS_HOME ortam değişkenini düzenleyerek.
  2. Web container'ınızı çalıştırmadan önce, JENKINS_HOME ortam değişkenini ayarlayarak.
  3. (Önerilmez) jenkins.war içerisindeki web.xml dosyasını değiştirerek (veya web container içerisinde açılmış halde bulunan halinde bu düzenlemeyi yapabilirsiniz).
Bu değer, Jenkins çalışırken değiştirilemez. Daha çok yaptığınız değişikliğin olup olmadığını görebilmeniz içindir.
